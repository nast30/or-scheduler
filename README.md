Runnigng the program:
1. The pom.xml file contains all the dependencies, including maven. After installing all dependencies with maven, the program is expected to run.
2. The application file is called OrSchedulerApplication. To execute the first call follow: http://localhost:8080/operation-room/doctor/1/get 
3. The data of doctor is hard coded in the DoctorService.class and the rooms data is hardcoded in the OperationRoomService.class
4. During the uptime of the app, there's a print to the console everytime the schedule is updated with the names of the doctors and the queue.
5. The logic of the scheduling is in the OperationRoomSchedulerService.class
